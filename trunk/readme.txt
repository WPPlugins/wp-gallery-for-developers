A plugin to add gallery metabox to post types for theme developers to easily display list of images on posts, pages or any post type you like.

== Description ==

The purpose of this plugin is to make WP theme developers life easier.

= Features =
* Add gallery metabox to any WP post type you like.
* Based on WP Media Library modal window. No learning, use what you are used to.
* Simple PHP code snippet to implement in your theme.
* Optional. Shortcodes.
* Optional. You can use your Flickr account as a gallery source for your images.

No clutter, no JS libraries, no CSS styles. It is all up to you!

== Installation ==
Upload plugin to your plugins folder (unzip) and activate it in WP admin.
